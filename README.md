# Personal Library Manager 📚

Web aplikacija za upravljanje ličnom bibliotekom, razvijena kao projekat u okviru predmeta **Web programiranje 1** na Akademiji strukovnih studija Šumadija.

## 🚀 Tehnologije
* **PHP & Laravel** - Backend logika i upravljanje rutama
* **Blade (Laravel View Engine)** - Kreiranje dinamičkih templejta
* **MySQL** - Upravljanje bazom podataka (knjige, autori, žanrovi)
* **Bootstrap 5 & CSS3** - Stil

## ✨ Funkcionalnosti
* **CRUD operacije:** Dodavanje, pregled, izmena i brisanje knjiga iz baze.
* **Kategorizacija:** Organizacija naslova po žanrovima i autorima.
* **Pretraga:** Brzo pronalaženje knjiga u bazi.
* **Validacija:** Siguran unos podataka putem Laravel formi.

## 🛠️ Instalacija (za lokalno pokretanje)
1. Klonirajte repozitorijum.
2. Pokrenite `composer install`.
3. Pokrenite migracije: `php artisan migrate`.
4. Pokrenite server: `php artisan serve`.
